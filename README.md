# 🤖 AI Chatbot

A conversational AI chatbot built with Python as my first project.
Powered by Groq API and LLaMA 3.3 model.

## 💡 Features
- Rule based responses (Level 1)
- Dictionary based responses (Level 2)
- AI powered conversations using Groq API (Level 3)
- Remembers conversation history
- Secure API key handling with .env file

## 🛠️ Tech Stack
- Python 3.11
- Groq API
- LLaMA 3.3 70B model
- python-dotenv

## ⚙️ How to Run

### 1. Clone the repository
git clone https://github.com/YOURUSERNAME/my-chatbot.git
cd my-chatbot

### 2. Create virtual environment
python -m venv .venv
source .venv/bin/activate

### 3. Install dependencies
pip install groq python-dotenv

### 4. Add your API key
Create a .env file and add:
GROQ_API_KEY=your_groq_api_key_here

### 5. Run the chatbot
python ai_chatbot.py
